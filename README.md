# Portfólio Profissional - MVP

Este projeto é um MVP de portfólio profissional desenvolvido com HTML5, CSS3 e JavaScript.

## Estrutura do projeto

- `index.html` -> estrutura principal da página
- `style.css` -> estilização e responsividade
- `script.js` -> renderização dinâmica dos projetos

## Como publicar no GitHub

1. Crie um repositório no GitHub.
2. Clique em **Add file** > **Upload files**.
3. Envie todos os arquivos desta pasta.
4. Clique em **Commit changes**.

## Como publicar no GitHub Pages

1. Abra o repositório no GitHub.
2. Vá em **Settings**.
3. Clique em **Pages**.
4. Em **Branch**, escolha **main** e depois **/root**.
5. Clique em **Save**.
6. Aguarde gerar o link do site.

O link ficará parecido com:

`https://seu-usuario.github.io/nome-do-repositorio`
